# Design System — App Gym

Based on **interface-design** (Precision & Density) for dashboards and admin/teacher panels.

## Direction

- **Personality:** Precision & Density
- **Foundation:** Cool (slate)
- **Depth:** Borders-only (no heavy shadows)

## Tokens

### Spacing (4px base)

Scale: 4, 8, 12, 16, 24, 32 → Tailwind: 1, 2, 3, 4, 6, 8

### Radius

Scale: 4px, 6px, 8px (sharp, technical)

### Typography

- Scale: 11, 12, 13, 14 (base), 16, 18
- Weights: 400, 500, 600

### Patterns

- **Card:** border faint, padding 12px (p-3), radius 6px (rounded-md), no shadow
- **Button:** compact height 32px (h-8), padding 8px 12px (px-3 py-2), radius 4px (rounded)
- **Table cell:** padding 8px 12px (p-3), text 13px (text-[13px])
- **Nav:** compact items, clear active state
